Unidades CSS en Acción

Una demostración de unidades de medida relativas y absolutas en CSS

Unidades Relativas

Las unidades relativas cambian basadas en otros factores como el tamaño de fuente del elemento padre o el tamaño del viewport.

Ejemplos: em, rem, %, vh, vw, vmin, vmax, ex, ch

Unidades Absolutas

Las unidades absolutas tienen un valor fijo independiente de cualquier otro factor. Son útiles cuando se conoce el medio de salida.

Ejemplos: px, pt, pc, in, cm, mm

Aprende más

Mejores Prácticas

En diseño web moderno, es recomendable usar principalmente unidades relativas para crear layouts flexibles y accesibles.

Las unidades absolutas son más útiles para impresión.

Unidades Utilizadas en Esta Página

  • rem: Relativo al tamaño de fuente del elemento raíz (html)
  • em: Relativo al tamaño de fuente del elemento padre
  • %: Porcentaje relativo al elemento padre
  • vh/vw: Porcentaje relativo a la altura/anchura del viewport
  • vmin: El menor valor entre vh y vw
  • ex: Relativo a la altura de la letra 'x' de la fuente actual
  • ch: Relativo al ancho del carácter '0' de la fuente actual
  • px: Píxeles (unidad absoluta para pantallas)
  • cm: Centímetros (unidad absoluta física)
  • mm: Milímetros (unidad absoluta física)
  • in: Pulgadas (unidad absoluta física)
  • pt: Puntos (unidad absoluta, tradicionalmente para impresión)
